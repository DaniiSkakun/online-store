## Selenium E2E (UI-visible)

Стартовый каркас для ручного/авто прогона с визуализацией браузера.

### Предварительно
- Node.js 18+
- Chrome/Edge установлен

### Установка
```bash
npm install
```

### Запуск теста (открытый браузер)
```bash
npm test
```

### Что есть
- `tests/smoke.spec.js` — пример открытия главной страницы (`http://localhost:3000`) с явным ожиданием и скриншотом.

### Настройка
- URL задаётся через env `BASE_URL` (по умолчанию `http://localhost:3000`).
